# Design System: MPS Site Relaunch (Iteration 2)

**Project ID:** 14017426173337939993

## 1. Visual Theme & Atmosphere

**Vibe:** Sophisticated, Deep, Metallic, Industrial-Futurism.
**Atmosphere:** Deep ocean or deep space. High-gloss reflective surfaces, metallic sheens, and fluid abstract forms. Less "Tron neon", more "luxury automotive" or "premium fintech".

## 2. Color Palette & Roles

### Basics

- **Deep Void Black (#020408)**: Main page background.
- **Midnight Blue (#051025)**: Secondary background / deep gradients.
- **Glass Base (rgba(5, 20, 45, 0.4))**: Background for cards/tiles.
- **Chrome Border (rgba(255, 255, 255, 0.2))**: Sharp, reflective borders.

### Accents

- **Electric Cobalt (#2563EB)**: Primary interaction color. Deep and rich.
- **Metallic Cyan (#60A5FA)**: Highlights and rim lighting (lighter than the base).
- **Silver/White (#F8FAFC)**: Text and highest highlights.

## 3. Typography Rules

- **Family:** 'Space Grotesk' (Headings), 'Inter' (Body).
- **Headings:** Large, crisp, tracking set slightly wide.

## 4. Component Stylings

- **Buttons:**
  - High-gloss finish.
  - Gradient backgrounds (Deep Blue to Cobalt).
  - Inner shadows to create depth (not flat).
- **Cards (Glass Tiles):**
  - Heavier blur (`backdrop-filter: blur(40px)`).
  - Reflective top border (gradient border).
  - Shadow: `0 20px 50px -10px rgba(0,0,0,0.5)`.

## 5. Layout Principles

- **Fluidity:** Elements should feel like they are floating in a viscous medium.
- **Depth:** Extreme use of Z-axis.
